It is made with html, css, javascript, you dont need credential id to access this.

## Run Locally

Clone the project

```bash
  git clone https://github.com/rzkiypratama/discount-calculator
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
